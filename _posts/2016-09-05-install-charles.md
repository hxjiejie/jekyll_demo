1. 下载

    https://www.charlesproxy.com/latest-release/download.do（选择 Linux (tar.gz, 12 MB) ）

2. 到下载目录，解压：

   sudo tar -xvf charles-proxy-4.0.tar.gz

3. 进入目录：

   cd charles/bin/

4. 安装java-jre

   sudo apt-get update

   sudo apt-get install default-jre

5. 运行 charles

   sudo ./charles
